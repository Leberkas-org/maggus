What this mockup shows

  • Left pane — persistent list of all features/bugs with approval status. Arrow key navigation, enter to inspect tasks.
  • Right pane — 4 tabs, switch with number keys 1–4. Tab 1 (Output) shown active here.
  • Features and bugs are separated by a visual divider in the list.
  • Approval state shown inline: ✓ appr / pending.
  • Reorder via alt+↑↓ (UI-only, no file changes).
